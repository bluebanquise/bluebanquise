Conman
------

Description
^^^^^^^^^^^

This role provides a conman daemon that logs ipmi serial consols.

Instructions
^^^^^^^^^^^^

To login into a console, use:

.. code-block:: bash

  conman mynode

To exit, simply press **&** then **.** .
