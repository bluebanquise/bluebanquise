BlueBanquise
------------

Description
^^^^^^^^^^^

This role install Ansible and dependencies on BlueBanquise controllers.
Mainly used for bootstrapping the stack or multi iceberg.

Instructions
^^^^^^^^^^^^

None

Input
^^^^^

None

Output

* python netaddr and jmespath
