Set Hostname
-------------------

Description
^^^^^^^^^^^

This role simply set a node hostname.

Instructions
^^^^^^^^^^^^

It is possible to ask for hostname used to be FQDN, by setting variable
**set_hostname_fqdn** variable to *true*.
