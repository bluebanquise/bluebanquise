DNS client
----------

Description
^^^^^^^^^^^

This role provides a basic /etc/resolv.conf file.

Instructions
^^^^^^^^^^^^

Configuration is made in group_vars/all/general_settings/network.yml.

It is possible to add here external DNS servers for clients.
