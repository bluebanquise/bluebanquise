Hosts file
----------

Description
^^^^^^^^^^^

This role provides a basic /etc/hosts files.

Instructions
^^^^^^^^^^^^

External hosts in /etc/ansible/all/general_settings/external_hosts.yml are added automatically into this hosts file.
