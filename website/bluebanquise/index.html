BlueBanquise is a project that aims to provide resources to deploy Linux based clusters of servers.

BlueBanquise


A project that provides basic resources to understand and deploy clusters of servers, with a strong focus on modularity and accessibility.
It contains 3 axes:
    - Ansible collections
    - Tutorials
    - Stack
This is an opensource and non profit project, not bounded to any company.

Ansible collections


BlueBanquise provides multiple Ansible collections.
The main collection, Infrastructure, is generic and made to deploy any kind of servers clusters, while other collections are made to specialize the cluster: High Performance Computing, Kubernetes, rendering farm, etc.

Tutorials


One of the main BlueBanquise's objective is to help junior system administrators to level up and gather enough experience to deploy manually Linux servers and understand what happens behind the hood.
Multiple generic tutorials are provided with a focus on accessibility and understanding.

Stack


The BlueBanquise stack is made to simplify provisioning of clusters, by acting as an abstraction layer between system administrators and the target cluster.
Stack provides interfaces to populate Ansible inventory, power manage servers, and schedule provisioning.
